Lights, Camera, Action!

You're looking at the content of the welcome component located at ui/welcome.reel. Referred to as a component, this directory encapsulates the HTML, CSS, JavaScript, and any other bundled resources it needs to do its thing.

Add a child component

  1. A component is made up of other child components introduced through a declaration in the parent's HTML template. Populate the existing entry for the main component inside ui/welcome.reel/welcome.html with the following: "main": { "prototype": "ui/main.reel", "properties": { "element": {"#": "main"} } } This will put an instance of the main component as a child of the welcome component attached to the element with the data-montage-id attribute of main.
  2. Refresh this page

Create your own components

  1. cd {{name}}
  2. minit create:component -n new-component
  3. Use your new component just like you would any other

Load your actual application

  1. Point the index.html's loader component at the main component directly by removing references to the welcome component. "owner": { "prototype": "montage/ui/loader.reel" } By default the loader will load the ui/main.reel component.
  2. Delete ui/welcome.reel

Run tests

  1. Install development dependencies npm install
  2. Run your tests
